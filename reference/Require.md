# Load a package by installing it if necessary

It is easier to run a script if we know the packages will be installed
if this additional step is necessary. Installation uses the RStudio
mirror of CRAN.

## Usage

``` r
Require(pkg)
```

## Arguments

- pkg:

  the package to be loaded/installed.

## Value

logical: to say that the package has been successfully loaded
(invisible)
