# Package index

## Explore Graphs

Explore ggplot graphs in accessible ways.

- [`VI()`](http://ajrgodfrey.github.io/BrailleR/reference/VI.md) :
  Extract information from a graphical object
- [`sort(`*`<VIgraph>`*`)`](http://ajrgodfrey.github.io/BrailleR/reference/sort.VIgraph.md)
  : Sort VI.ggplot points list
- [`grep(`*`<VIgraph>`*`)`](http://ajrgodfrey.github.io/BrailleR/reference/grep.VIgraph.md)
  [`gsub(`*`<VIgraph>`*`)`](http://ajrgodfrey.github.io/BrailleR/reference/grep.VIgraph.md)
  : String manipulation of the output produced by VI.ggplot
- [`Describe()`](http://ajrgodfrey.github.io/BrailleR/reference/Describe.md)
  : Describe method to help visualize plot types.
- [`MakeAccessibleSVG()`](http://ajrgodfrey.github.io/BrailleR/reference/MakeAccessibleSVGMethod.md)
  : Create and display a accessible exploration webpage SVG of graph
- [`BrowseSVG()`](http://ajrgodfrey.github.io/BrailleR/reference/BrowseSVG.md)
  : Merge svg and xml file in html file
- [`SVGThis()`](http://ajrgodfrey.github.io/BrailleR/reference/SVGThis.md)
  : Save commonly used graphs as structured SVG files.
- [`ViewSVG()`](http://ajrgodfrey.github.io/BrailleR/reference/ViewSVG.md)
  : Create the necessary files to allow interactive viewing of SVG
- [`AddXML()`](http://ajrgodfrey.github.io/BrailleR/reference/AddXMLMethod.md)
  : Create XML files to sit alongside SVG files in order to make an
  accessible graph experience.
- [`WTF()`](http://ajrgodfrey.github.io/BrailleR/reference/WTF.md)
  [`wtf()`](http://ajrgodfrey.github.io/BrailleR/reference/WTF.md) :
  What's this figure?
- [`summary(`*`<recordedplot>`*`)`](http://ajrgodfrey.github.io/BrailleR/reference/summary.recordedplot.md)
  : Summarise the display list for a graphics device

## Graph and analysis shortcuts

Functions to help speed up graph making and analysis tasks

- [`boxplot()`](http://ajrgodfrey.github.io/BrailleR/reference/boxplot.md)
  : Create a standard boxplot with a few extra elements added to the
  output object
- [`dotplot()`](http://ajrgodfrey.github.io/BrailleR/reference/dotplot.md)
  : create a dotplot using stripchart
- [`hist()`](http://ajrgodfrey.github.io/BrailleR/reference/hist.md) :
  Create a standard histogram with a few extra elements added to the
  output object
- [`OneFactor()`](http://ajrgodfrey.github.io/BrailleR/reference/OneFactor.md)
  : Analysis for a continuous response for one group factor
- [`OnePredictor()`](http://ajrgodfrey.github.io/BrailleR/reference/OnePredictor.md)
  : Exploration of the relationship between a response and a single
  predictor
- [`ScatterPlot()`](http://ajrgodfrey.github.io/BrailleR/reference/ScatterPlot.md)
  [`FittedLinePlot()`](http://ajrgodfrey.github.io/BrailleR/reference/ScatterPlot.md)
  : Create a standard scatter plot with a few extra elements added to
  the output object
- [`ThreeFactors()`](http://ajrgodfrey.github.io/BrailleR/reference/ThreeFactors.md)
  : A convenience function for a Three-way analysis
- [`TimeSeriesPlot()`](http://ajrgodfrey.github.io/BrailleR/reference/TSPlot.md)
  : Create a standard time series plot with a few extra elements added
  to the output object
- [`TwoFactors()`](http://ajrgodfrey.github.io/BrailleR/reference/TwoFactors.md)
  : A convenience function for a two-way analysis
- [`UniDesc()`](http://ajrgodfrey.github.io/BrailleR/reference/UniDesc.md)
  : Descriptive statistics and graphs for univariate data
- [`WhereXY()`](http://ajrgodfrey.github.io/BrailleR/reference/WhereXY.md)
  : Count points in a scatter plot
